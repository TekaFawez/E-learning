La propriété font-

Maintenant que vous savez coder en HTML et CSS, nous allons voir l'ensemble des propriétés et balises possibles.

FONT-FAMILY

L'un des éléments le plus important du webdesign est la ou les typos utilisées. Historiquement, il n'existe que très peu de polices utlisables sur un site internet :

  • Arial
  • Times
  • New Roman
  • Palatino
  • Garamond
  • Bookman
  • Avant Garde
  • Courier
  • Courier New

FONT-WEIGHT

La propriété font-weight permet de contrôler la graisse de notre texte.

Voilà la liste des valeurs possibles :

  • normal : Niveau de graisse regular.
  • bold : Niveau de graisse gras.
  • lighter : Diminue d'un niveau la valeur de graisse.
  • bolder : Augmente d'un niveau la valeur de graisse.

FONT-STYLE

La propriété font-style permet d'utiliser une version italique de la police de caractère où, à défaut, de pencher artificiellement le texte.

FONT-SIZE

La propriété font-size permet de définir une taille à notre texte. Il est possible d'utiliser plusieurs unités :

  • Les pixels : ils permettent de définir une valeur fixe, absolue et identique sur tous les navigateurs.
  • Les ems : ils permettent de définir une valeur dynamique basée sur la taille de l'élément parent.
  • Les mots clefs : comme les pixels, ils permettent de définir une valeur fixe, absolue et identique sur tous les navigateurs.
  • Les rems : ils permettent de définir une valeur dynamique basée sur la taille de l'élément html

TEXT-DECORATION

La propriété text-decoration permet de décorer le texte avec une ligne placée sous, sur ou en travers le texte.

Nous pouvons agir sur 3 propriétés :

  • Le type de ligne (sous, sur ou en travers),
  • Le style de la ligne (pointillée, pleine, …),
  • La couleur de la ligne.
